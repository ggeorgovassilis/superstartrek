About superstartrek

This is a port of Super Star Trek to mobile phone browsers. This port takes several liberties:

  • The UI is directed at touch/click operation and is not command based as in the original game
  • Much of the original game's brilliance lies in estimating consumption, damages and distances. In a GUI this doesn't make much sense, hence this port is exploring different ways of diversifying game play
  • This is work in progress
